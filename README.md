# PLORD - Resin
Plastordance (Plastic Ordance) modification for Factorio

This is a mod, which will add many resources, crafts and other interesting things to the game.

In plan I have to add many chemical elements and compounds to diversify the game experience.

Current stage - mod prototyping, adding the resin and related crafts (making rubber, refined resin, melted resin and other).

# Resin / Смола
![Resin-ore icon](https://raw.githubusercontent.com/DemerNkardaz/PLORD/misc/icns/resin-ore.png?raw=true)<br><br>
This is a new resource what you can to collect from trees, stones and you can find some ore-like resin deposits, which able to harvested by drills.<br>
With unlock starting tecnhology "Resin Harvesting" give you ability to refine wood to resin. It some risky - you have chance to nothing result with spending of wood.<br>
Resin is will be important resource for some crafts in later updates.<br>
All resins can be consumed as a fuel, but this is a bad idea - very low power gain and high-rate consuming of item. However, it can be fixed with some complex crafts to useful fuel recipe with coal's derivative...<br><br>
Это новый ресурс, который можно добывать срубая деревья, копая булыжники, а также можно обнаружить рудоподобные залежи смолы, на которые можно ставить буры.<br>
С исследованием стартовой технологии "Добыча смолы" вам даётся способность перерабатывать древесину в смолу. Это рисковано - всегда есть шанс потратить древесину без результата.<br>
Смола станет важным ресурсом для ряда крафтов в будущих обновлениях.<br>
Все смолы могут быть потрачены как топливо, но это плохая идея - очень низкая энергоотдача и высокой расход предмета. Однако, это может быть исправлено некоторыми комплексными крафтами для рецепта полезного топлива с применение производного угля...<br><br>


# Melted Resin / Расплав смолы

Game vanilla recipes will be changed to use new resources, as example - belts will use rubber.

And will added new useful recipes - new barreled fuel that crafts from refined resin and coal or coal's derivative.

Refined resin you can craft from melted resin, for it I will add special "Chemical Boiler", consuming electricity and liquidify x50 resin to x100 melted resin.
